Einleitung

Mit Hilfe dieses Programmes können Sie zwei Ordner z.B. auf einem Notebook und einer Arbeitsstation abgleichen. Getestet und entwickelt wurde diese Anwendung unter Mac OS X 10.4, SuSe Linux 10.1, Kubuntu 6.06, Windows XP und Windows 2000.

Unter Linux und Windows müssen Sie die Sun Java Laufzeitumgebung in der Version 1.4.x oder neuer installiert haben. Achten Sie auch darauf, daß die Javaumgebung in Ihrer Pfadvariable eingetragen ist. Mac OS X bringt automatisch eine solche mit.

Starten

Zum Starten des Programmes gehen Sie wie folgt vor:

Mac OS X Finder:

Starten Sie das Programm wie gewohnt durch Doppelklick auf das Symbol für "XJ Sync".

Darwin/Linux:

Wechseln Sie in das Verzeichnis "XJ Sync.app" und starten Sie das Shellscript xjsync. Möglicherweise müssen Sie mit dem Kommando

chmod +x xjsync

das Skript ausführbar machen.

Windows:

Öffnen Sie das Verzeichnis "XJ Sync.app" und starten Sie von dort das Shellscript xjsync.bat. Im Verzeichnis XJ Sync.app\Contents\Windows gibt es auch eine ausführbare Datei xjsync.exe, die sofern Sie über die entsprechenden Rechte verfügen, eine Verknüpfung zu den Datenbankdateien von XJ Sync herstellen kann.

Anwendung

Die Anwendung dieses Programmes sollte eigentlich sehr einfach sein, weshalb hier nur kurz darauf eingegangen werden soll:

Zum Erstellen einer neuen Datenbank wählen Sie zwei Verzeichnisse aus, die abgeglichen werden sollen. Anschließend werden diese sofort verglichen und das Ergebnis angezeigt. Nun können Sie noch wählen auf welche Art und Weise die beiden Verzeichnisse abgeglichen werden sollen. Dabei haben Sie die Wahl zwischen "Beide Seiten", "Von Links nach Rechts" und von "Rechts nach Links" aus dem Menü "Quelle/Ziel".

Im Menü Operation können Sie für ausgewählte Dateien die berechnete Operation selbst vergeben. Darüber hinaus können Sie dort einen oder mehrere durch Strichpunkt getrennte Filter festlegen, mit denen bestimmte Dateien ausgeblendet werden sollen.
Beispiel: *.obj;*.csm;*.class

Nach dem Klick auf das Diskettensymbol wird die Datenbank gespeichert und kann jederzeit wieder geladen werden indem Sie den Schalter "Öffnen..." drücken. Schließlich wird mit dem Symbol für die Synchronisation der Abgleich gestartet.
